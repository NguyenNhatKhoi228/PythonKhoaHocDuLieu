# PythonKhoaHocDuLieu

## Dữ liệu
Link dataset (từ Kaggle): https://www.kaggle.com/code/gpreda/credit-card-fraud-detection-predictive-models/input?select=creditcard.csv

> Lưu ý: do dữ liệu được lưu trên Kaggle, bạn cần tải file `creditcard.csv` về máy và đặt vào cùng thư mục với notebook hoặc cập nhật đường dẫn trong notebook tương ứng.

## Yêu cầu môi trường
- Python 3.8+
- Thư viện gợi ý: pandas, numpy, scikit-learn, matplotlib, seaborn, imbalanced-learn, jupyter

Cài đặt nhanh (nếu có file `requirements.txt`):

```
pip install -r requirements.txt
```

## Cách chạy
1. Mở terminal, chuyển vào thư mục repo.
2. Chạy `jupyter lab` hoặc `jupyter notebook`.
3. Mở các notebook (*.ipynb) để xem các bước tiền xử lý, phân tích và mô hình hóa.

## Nội dung chính
- Các notebook trình bày khám phá dữ liệu, tiền xử lý, cân bằng lớp, huấn luyện và đánh giá mô hình để phát hiện gian lận.
